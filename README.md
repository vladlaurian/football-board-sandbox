# Football Board Sandbox v3.4

- Terenul este centrat by default pe PC și pe iPad.
- Pan-ul nu mai este salvat în cloud, ca să nu strice poziția între dispozitive.
- Poți muta terenul pe desktop prin drag pe gazon/fundal.
- Pe touch rămâne pan/zoom cu două degete.
- Piesele și mingea rămân la dimensiunea vizuală normală.
