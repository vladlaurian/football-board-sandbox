# Football Board Sandbox v1.3

## v1.3
- repoziționare default albastru 4-4-2:
  - LM/RM mai lateral și mai sus
  - CM-urile o căsuță mai în spate
  - ST-urile în afara cercului central
- repoziționare default roșu 4-2-3-1:
  - CDM-urile 2 căsuțe mai în spate
  - LWB/RWB 2 căsuțe mai lateral
  - AM o căsuță mai în spate
  - LW/RW mai lateral și mai sus
  - ST pe linie cu LW/RW
- punctul de centru are buffer verde și nu mai este deformat de linia de centru
